Guided Sigma Builder

Use this page to turn a selected Windows event into a Sigma draft with deterministic assistance instead of starting from raw YAML.

  • Visible Evidence Context keeps the selected event, key fields, and raw JSON visible while the analyst builds the rule.
  • Case Support Context shows channel counts, top EventIDs, and observed EventData keys so the analyst can judge rarity and stability.
  • Selectable Detection Atoms turn EventID, logsource service, and EventData values into clickable Sigma selector candidates.
  • Override Controls let the analyst change the Sigma field, operator, and value directly in each card before carrying it into the preview.
  • Live Rule Preview renders the selected atoms into Sigma YAML automatically.
  • Deterministic Validation continues in the manual editor, where the analyst can test the Sigma draft against the current case's Windows event records before saving it.
