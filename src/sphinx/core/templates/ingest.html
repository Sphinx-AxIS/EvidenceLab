Ingest Evidence

Upload evidence files to add them to the current case. The system parses the data, stores it as structured records, and automatically extracts entities (IPs, domains, hashes, etc.).

Supported Formats

  • PCAP / PCAPNG — Network capture files. Automatically processed by Suricata (IDS alerts), Zeek (connection metadata, protocol logs), and tshark (TCP stream reconstruction). Select "PCAP Upload" and choose your capture file.
  • JSON / JSONL — Structured log files. Select the appropriate record type (e.g., win_evt_security for Windows Security logs) and upload a JSON array or newline-delimited JSON file.

What Happens During Ingest

  • PCAP ingest runs in the background — you can monitor progress on the Dashboard.
  • JSON ingest is immediate — records are inserted and you're redirected back with a count.
  • Deployed detection rules (Suricata and Sigma) automatically run against newly ingested evidence.
