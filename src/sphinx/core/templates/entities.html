Entities

Entities are indicators of compromise (IOCs) automatically extracted from evidence records using pattern matching. They help you quickly identify IPs, domains, emails, users, and hashes involved in the incident.

Entity Types

  • ip — IPv4 addresses found in evidence.
  • domain — Domain names (e.g., evil.domain.com).
  • email — Email addresses.
  • user — Username patterns (DOMAIN\user format).
  • url — Full URLs.
  • hash_md5 / hash_sha1 / hash_sha256 — File hashes.

Pivot Analysis

  • Click Pivot on any entity to see all records that contain it and which other entities co-occur in those same records.
  • This reveals relationships — e.g., pivoting on an IP shows you every connection, alert, and log entry involving that IP, plus the domains, users, and hashes associated with it.
