Guided Suricata Builder

Use this page to turn a selected packet-derived record into a Suricata draft without starting from raw syntax.

  • Flow Review keeps the key packet and stream facts visible.
  • Detection Strategy lets the analyst choose IOC, behavior, or service-focused logic before selecting anchors.
  • Content Match Style lets the analyst keep simple literal content matches or switch to resilient PCRE when stream reassembly matters.
  • Candidate Detection Atoms show High, Medium, and Low options that can be added to the draft with a click.
  • Frame Provenance shows whether a candidate string was seen inside one frame or only after stream reassembly.
  • Live Rule Preview renders the selected facts into Suricata syntax automatically.
  • Warnings call out brittle choices such as exact IPs in behavior mode or content strings that are too short.
  • Raw Data (JSON) and payload previews are rendered as escaped literal text so hostile evidence content cannot break the page layout.
