Create Detection Rule

Manually write a Suricata or Sigma detection rule. You can also paste rules from external sources.

Suricata Rules

  • One rule per line in standard Suricata syntax.
  • Use $HOME_NET and $EXTERNAL_NET variables, not literal IPs.
  • Include: msg, content/pcre matchers, classtype, sid, rev.
  • SID range for custom rules: 9100000+.

Sigma Rules

  • Write in YAML format following the Sigma specification.
  • Include: title, logsource, detection, condition, level.
  • When deployed, the rule is compiled to SQL and run against ingested Windows event logs.
